Login / Signup

Sign up from enquiry, then access with mobile and reference ID.

Students first enquire about a program. After submitting the enquiry form, they receive an email PDF receipt with a reference ID. The registered mobile number and that reference ID become the access details for student support.

Reference format Recommended format: DD-MM-YYYY-MobileNumber, for example 14-07-2026-8088835686.

Exact procedure

How a student signs up and requests access.

This keeps the public flow simple while the backend checks Google Sheet data for mobile number and reference ID matching.

01

Explore programs

Student or parent checks available programs and chooses the suitable course.

02

Submit enquiry

Student fills the enquiry form with name, email, mobile number, location, and interested program.

03

Receive email receipt

System stores the response in Google Sheet and sends a PDF receipt with reference ID.

04

Enter access details

Student enters the same registered mobile number and enquiry reference ID on this page.

05

Backend verifies

Backend matches mobile plus reference ID in the response sheet and rejects duplicate mobile registrations.

06

Open dashboard

After verification, the student can land on their own course, class, homework, marks, and update page.

Student access request

Verify with registered mobile and reference ID.

Enter the same mobile number used in the enquiry form and the reference ID from the email PDF. Until automatic backend login is deployed, this prepares a WhatsApp verification request for the team.

Signup rules

One mobile number, one registered student profile.

The enquiry response becomes the first student record. The same mobile number is treated as the username, and the generated reference ID proves which enquiry belongs to that student.

01Reference ID format: DD-MM-YYYY-MobileNumber.
02If the mobile number already exists in the response sheet, backend should reject duplicate signup.
03If mobile number and reference ID match, the student can be sent to their dashboard.
