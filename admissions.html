Process

Four steps from interest to admission.

This public flow keeps families clear on counselling, recommendation, payment guidance, and batch confirmation.

  1. Submit interestShare student name, contact details, class, and interested program.
  2. Counselling callDiscuss learning goals, current class level, batch timing, and expectations.
  3. Program recommendationRecommend basic foundation, Classes 1-3, Classes 4-5 competitive, Classes 6-10 Maths, or +1/+2 Physics and Maths.
  4. Batch confirmationConfirm documents, fee plan, timing, and start date when available.
